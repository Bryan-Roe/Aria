# Web Pages Quick Reference

## Start Here 🚀

### Main Entry Point
- **Web Hub**: `http://localhost/web-hub.html` - Central navigation hub

### Three Main Areas

#### 1. Aria Interfaces (Character & Chat)
- Aria Visual Commands: Real-time character control system
- Chat Interface: Multi-provider AI chat with Aria
- Auto-Execute Planner: Command planning tool

#### 2. Quantum Experiences (Simulations & Worlds)
- Quantum Portal: Hub for quantum experiences
- 2D World: Interactive quantum simulation
- 3D World: 3D canvas visualization
- Features & Testing: Feature guide and test suite

#### 3. Dashboards & Monitoring (Training & System)
- Training Dashboard: Real-time metrics
- Command Center: System control hub
- Model Analysis: GGUF visualization
- Advanced Analytics: Optimization tools

- 🟢 **Online/Available** - Service is running
- 🟡 **Checking** - Connection test in progress
- 🔴 **Offline/Unavailable** - Service not responding
- 💫 **Connected** - API connected and ready
