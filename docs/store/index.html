Frequently Asked Questions

Can I use Aria without any API keys?

Yes! The Free tier uses a local model fallback that requires no external API keys or accounts. You can interact with Aria's character, use the chat CLI, and explore the platform core features.

What AI providers are supported?

Aria supports Azure OpenAI, OpenAI, LMStudio, LoRA adapters (custom fine-tuned models), and a zero-dependency local fallback. Provider detection is automatic with configurable fallback chains.

Do I need a GPU for training?

For LoRA fine-tuning, a CUDA-capable GPU is recommended but not required. CPU training works for small datasets. Enterprise plans include access to dedicated Azure GPU resources.

How does quantum ML work?

Aria integrates Qiskit and PennyLane for hybrid quantum-classical machine learning. You can simulate circuits locally, use the Azure Quantum simulator, or submit jobs to real quantum hardware (Enterprise only).

Is the platform open source?

Yes! Aria is fully open source on GitHub. The Free tier gives you access to all source code and local features. Paid plans add cloud infrastructure, managed services, and premium support.
