Get Started

Up and running in minutes

1
Clone
git clone https://github.com/Bryan-Roe/Aria
2
Install
pip install -r requirements.txt
Python 3.9+ required
3
Configure
Copy local.settings.json.example and add API keys (optional — local mode needs none)
4
Launch Aria
cd apps/aria && python server.py
Visit localhost:8080
5
Start APIs
func host start
All endpoints on port 7071
6
Validate
python scripts/fast_validate.py
Quick health check
