🚀 Platform Features

  • Interactive 3D animated character with natural language control
  • Multi-provider AI chat (Azure OpenAI, OpenAI, LMStudio, Local)
  • LoRA fine-tuning for Phi-3.5 models
  • Quantum-enhanced machine learning with Azure Quantum
  • Autonomous training orchestration with adaptive epochs
  • Real-time monitoring and analytics dashboards
  • Object interaction system with physics
  • Azure Functions integration for serverless deployment

🔧 Local Development

  • Clone the repository from GitHub
  • Install Python 3.9+ and dependencies
  • Configure API keys for Azure OpenAI or OpenAI (optional)
  • Start web servers: aria_web/server.py, dashboard/app.py
  • Run Azure Functions locally with 'func start'
  • Access full AI capabilities, quantum computing, and training
