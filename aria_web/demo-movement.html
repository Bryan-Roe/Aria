🎯 Aria Movement Patterns

Interactive demonstration of character movement algorithms

Linear Movement

Aria moves in a straight line from start to end point.

A
Smoothly interpolates position over time. Ideal for direct pathing.

Circular Orbit

Aria follows a circular path around a center point.

A
Uses trigonometry (sin/cos) for smooth circular motion.

Zigzag Pattern

Aria moves in a zigzag pattern across the stage.

A
Sequential waypoint following with sharp direction changes.

Spiral Descent

Aria follows an expanding spiral pattern.

A
Combines circular motion with radius expansion over time.

Wave Motion

Aria moves forward while oscillating up and down.

A
Uses sine waves for smooth vertical oscillation during forward movement.

Random Walk

Aria moves randomly around the stage.

A
Picks random waypoints and moves toward them autonomously.

📊 Algorithm Details

Linear: Lerp(start, end, t)
Circle: x = cx + r*cos(θ), y = cy + r*sin(θ)
Zigzag: Connect waypoints [0,0] → [100%,50%] → [0,100%]
Spiral: radius = t × max_r, angle = t × 4π
Wave: x = t, y = sin(t × freq) × amplitude
Random: Brownian motion with waypoint seeking
