# 🎯 Aria Demo System - Quick Start

Welcome to Aria's Interactive Demo Platform! This quick-start guide gets you up and running in 30 seconds.

## ⚡ Quick Launch (3 Ways)

### 1. **Main Aria Interface** (Recommended)
```
1. Open: aria_web/index.html
2. Scroll to: "🎯 Interactive Demos" section
3. Click: Any "Launch Demo" button
```

### 2. **Demo Launcher Dashboard**
```
Open: aria_web/demos.html
Choose demo from 6-button grid
```

### 3. **Demo Center Hub**
```
Open: aria_web/demo-center.html
Browse all demos with full descriptions
```

## 🎮 Interactive Examples

### Try Movement Demo
1. Open `demo-movement.html`
2. Click "Start" next to any movement type
3. Watch the character animate in real-time
4. Click "Reset" to restart

### Try Dialogue Demo
1. Open `demo-dialogue.html`
2. Type a message (or click quick-response button)
3. Press Enter or click "Send"
4. Aria responds with contextual dialogue
5. Switch dialogue mode: Casual → Teaching → Creative → Supportive

### Try Emotions Demo
1. Open `demo-emotions.html`
2. Drag slider to adjust emotion intensity
3. Watch emoji face change in real-time
4. Click preset buttons for quick expressions

### Try Learning Demo
1. Open `demo-learning.html`
2. Click "Start Training" for any skill
3. Watch progress bars fill as learning happens
4. View statistics update in real-time

### Try Quantum Demo
1. Open `demo-quantum.html`
2. Click "Observe" to create superposition
3. Click "Measure" to collapse state
4. Try quantum gates (Pauli-X, Hadamard)
5. Watch waveform visualization
