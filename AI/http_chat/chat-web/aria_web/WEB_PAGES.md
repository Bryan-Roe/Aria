# Aria Web Pages Index

A quick index of the HTML entry points and supporting docs in `aria_web/`.

## Pages (HTML)

- `index.html` - Main Aria Visual Command System UI
- `quantum-portal.html` - Portal hub for all Quantum World experiences
- `quantum-world.html` - Interactive 2D Quantum World (API-backed)
- `quantum-world-3d.html` - 3D canvas visualization (no API)
- `quantum-world-features.html` - Feature showcase and usage guide
- `test-quantum-world.html` - Test suite for the Quantum World (API-backed)
- `auto-execute.html` - Command planning/execution UI (API-backed)
- `test.html` - Misc test page

## API-backed pages (GitHub Pages friendly)

These pages accept `?api=https://your-server` to point at a remote backend:

- `quantum-world.html`
- `test-quantum-world.html`
- `auto-execute.html`
