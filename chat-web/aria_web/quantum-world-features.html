🎯 How to Use New Features

Interact with Objects:

• Click - View object details
• Shift+Click - Select object (golden outline)
• Drag - Move object around stage
• Delete key - Remove selected object
• Quantum objects spin when selected!

Control Animation:

Use the Animation Speed slider (0.5x to 3x) to control how fast objects float and pulse. Perfect for creating slow-motion quantum effects or fast-paced visualizations!

Save & Load Worlds:

1. Generate a world you like
2. Click 💾 Save button
3. Come back anytime and click 📂 Load
4. Your world will be restored with all objects in place!
