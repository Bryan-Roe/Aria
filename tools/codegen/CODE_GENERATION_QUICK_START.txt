# Code Generation Quick Reference Card

**For fast access to code generation in Aria**

## 🚀 Quickest Way: Copilot Chat

Press `Ctrl+Shift+I` and type:
```
@llm-maker Generate a function that [what you need]
```

**Examples:**
```
@llm-maker Generate a function that validates phone numbers
@llm-maker Generate a function that extracts URLs from text
@llm-maker Build a landing page website with pricing
```

---

## 📊 What Can Be Generated

### ✅ Functions (Safe)
- Email, phone, URL validation
- Data parsing (CSV, JSON, query strings)
- Text utilities (slugify, reverse, analyze)
- Math functions (primes, fibonacci, interest)
- String operations (camelCase, snake_case)
- List operations (flatten, chunk, rotate)

### ✅ Websites
- Portfolio sites
- Blog templates
- Landing pages
- Documentation sites
- E-commerce sites

### ❌ NOT Allowed
- Code using `os`, `sys`, `subprocess`
- Functions with `eval` or `exec`
- Network/file system access
- Dynamic imports

## ⚡ Pro Tips

### Tip 1: Be Specific
```
❌ Bad: "Generate a validator"
✅ Good: "Generate a function that validates US phone numbers (10 digits)"
```

### Tip 2: Include Domain Context
```
❌ Bad: "Calculate interest"
✅ Good: "Calculate compound interest with monthly compounding"
```

### Tip 3: Request Multiple at Once
```
✅ Good: "@llm-maker Generate these 3 utility functions: ..."
```

### Tip 4: Batch Similar Tasks
```
✅ Good: "Generate validators for email, phone, and credit card"
```

## 🐛 Troubleshooting

### "Import X is not allowed"
**Solution**: Request simpler version without file/system access
```
✅ Instead of: "Read CSV from file"
   Use: "Parse a CSV line string"
```

### "Code doesn't work"
**Solution**: Add more context or simplify
```
✅ Instead of: "Parse JSON"
   Use: "Parse simple key=value format"
```

### "Website missing files"
**Solution**: Start with fewer pages
```
✅ Instead of: "Build complex site with 10 pages"
   Use: "Create a simple landing page"
```
